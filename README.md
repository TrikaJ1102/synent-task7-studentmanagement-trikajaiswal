# Student Management System

## Objective

This project is a command-line based Student Management System developed using Python. It helps manage student records efficiently using file handling and CRUD operations.

---

## Features

- Add student records
- View student details
- Update student information
- Delete student records
- Persistent storage using JSON
- Error handling for invalid inputs

---

## Technologies Used

- Python
- JSON
- OS Module


## Project Structure

```plaintext
Task7_StudentManagement/
│
├── main.py           # Main application file
├── students.json     # Stores student records
├── README.md         # Project documentation
└── screenshots/      # Output screenshots
```


## How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

```bash
python main.py
```
